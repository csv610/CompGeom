# Stochastic PDEs on Meshes

## 1. Overview
Stochastic Partial Differential Equations (SPDEs) are PDEs that incorporate random terms, representing uncertainty, thermal noise, or heterogeneous materials. In computational geometry and mesh processing, SPDEs are used to model phenomena like texture synthesis, surface growth, and structural uncertainty. Solving SPDEs on meshes requires combining finite element methods (FEM) or discrete exterior calculus (DEC) with stochastic integration techniques.

## 2. Definitions
*   **Stochastic Term ($\xi(x, t)$)**: A random process, often modeled as Gaussian white noise.
*   **Mean and Variance**: The statistical properties of the field resulting from the SPDE.
*   **Wiener Process**: The continuous-time stochastic process used to model Brownian motion.
*   **Itô Calculus**: A branch of mathematics for integrating with respect to stochastic processes.

## 3. Theory
A common SPDE on a surface is the **Stochastic Heat Equation**:
$$\frac{\partial u}{\partial t} = \Delta u + \dot{W}$$
where $W$ is a space-time white noise. In the discrete setting on a mesh:
1.  Discretize the spatial operator using the **Cotangent Laplacian** $L$.
2.  Discretize the noise term $\dot{W}$ by assigning a random Gaussian value to each vertex, weighted by the square root of the vertex's mass (area): $w_i \sim \mathcal{N}(0, 1) / \sqrt{A_i}$.
3.  Integrate in time using a stochastic scheme like **Euler-Maruyama**.

Another important class is the **Stationary SPDE**, such as $( \kappa^2 - \Delta )^\alpha u = \xi$, which is used to generate **Gaussian Random Fields (GRFs)** with specific correlation structures (Matérn covariance) on complex manifolds.

## 4. Pseudo code
### Simulating a Gaussian Random Field (Matérn Type)
```python
function GenerateGaussianRandomField(mesh, kappa, alpha):
    # 1. Build discrete operators
    L, M = ComputeLaplacianAndMass(mesh)
    
    # 2. Construct operator (kappa^2 * M + L)
    # This is for alpha=1; for higher alpha, use powers of the operator
    Operator = kappa**2 * M + L
    
    # 3. Generate white noise vector
    # Noise variance must scale with the inverse of the area
    white_noise = np.random.normal(0, 1, size=num_vertices)
    rhs = M.sqrt() * white_noise
    
    # 4. Solve the linear system
    # Operator * u = rhs
    u = SolveSparseSystem(Operator, rhs)
    
    return u
```

## 5. Parameters Selections
*   **Correlation Length ($1/\kappa$ )**: Determines the spatial scale of the random features.
*   **Smoothness ($\alpha$ )**: Controls the differentiability of the resulting field.
*   **Time Step ($dt$ )**: Critical for stability in time-dependent SPDEs.

## 6. Complexity
*   **Assembly**: $O(F)$ where $F$ is face count.
*   **Solving**: $O(N^{1.5})$ for sparse linear systems.
*   **Multiple Realizations**: Once the operator is factorized (e.g., using Cholesky), generating new random samples is $O(N \log N)$ or $O(N)$.

## 7. Usages
*   **Texture Synthesis**: Generating natural-looking textures (e.g., marble, wood, clouds) directly on 3D surfaces.
*   **Uncertainty Quantification (UQ)**: Modeling the effect of manufacturing tolerances on the structural integrity of a part.
*   **Geophysics**: Modeling the distribution of minerals or seismic properties in the earth's crust.
*   **Biological Growth**: Simulating the stochastic growth of cells or tumors on an organ surface.
*   **Computer Graphics**: Adding realistic "jitter" or noise to animations and simulations.

## 8. Testing methods and Edge cases
*   **Flat Domain**: Verify that the correlation function of the generated field matches the analytical Matérn covariance on a 2D plane.
*   **Statistical Convergence**: Verify that the mean of many realizations approaches the solution of the deterministic version of the PDE.
*   **Mesh Independence**: Ensure that the statistical properties of the field are consistent across different mesh resolutions.
*   **Symmetry**: Check if the random field respects the geometric symmetries of the mesh (on average).
*   **Boundary Conditions**: Verify that Dirichlet or Neumann conditions are correctly enforced on the random field.
