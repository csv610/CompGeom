# `trianglemesh/walks.py`

Purpose: grid-walk simulations and space-filling curve helpers.

Algorithms:
- bounded random walks in 2D and 3D
- self-avoiding walk with trap detection
- Peano, Morton, and Hilbert index decoding
- zigzag and spiral grid traversal generation

Inputs: grid sizes, start positions, and curve parameters.

Outputs: path statistics, paths, or integer grid coordinates.
